# A股科技企业智能投研助理

## 项目简介

这是一个基于 LangGraph 构建的工业级 A 股科技企业智能投研助理，专为解决投资者在科技股投资中面临的信息过载、踩雷、错过机会、分析慢等核心痛点而设计。

## 核心功能

- **24小时盯盘/盯信息**：自动采集和分析市场信息
- **快速筛选有效信号**：从海量信息中提取关键信号
- **深度分析**：对财报、政策、资金流向等进行专业分析
- **风险预警**：及时识别潜在风险，避免踩雷
- **机会提示**：捕捉投资机会，不缺席主线
- **机构级投研简报**：生成专业、简洁的投研报告

## 技术架构

基于 LangGraph 构建的状态图工作流，包含以下核心节点：

1. **信息采集节点**：从官方公告、市场数据、研报和行业信息中采集数据
2. **信息清洗&去重节点**：过滤无效信息，标准化格式
3. **信息分类节点**：按财报/公告、政策/行业、资金/盘面、研报/纪要分类
4. **深度分析节点**：对不同类型信息进行专业分析
5. **多维度推理节点**：评估机会和风险评分
6. **决策分支节点**：根据评分区分高风险、高机会和中性信号
7. **生成投研简报节点**：生成机构级投研简报
8. **输出/推送节点**：推送到企业微信等指定渠道

## 技术栈

- **LangGraph**：构建状态图工作流
- **OpenAI API**：进行自然语言处理和分析
- **Python**：核心开发语言
- **企业微信 API**：实现消息推送
- **Pydantic**：数据模型定义

## 安装步骤

### 1. 克隆项目

```bash
git clone <repository_url>
cd invest
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置环境变量

复制 `.env.example` 文件为 `.env`，并填写相关配置：

```bash
cp .env.example .env
```

编辑 `.env` 文件，填写以下配置：

- **LLM 配置**：
  - `OPENAI_API_KEY` - OpenAI API 密钥
  - `OPENAI_BASE_URL` - API 基础 URL（如 https://api.openai.com/v1）
  - `OPENAI_MODEL` - 使用的模型（如 gpt-4）

- **数据源配置**：
  - `WIND_API_KEY` - Wind 数据 API 密钥
  - `IFIND_API_KEY` - iFinD 数据 API 密钥
  - `EAST_MONEY_API_KEY` - 东方财富 API 密钥
  - `SINA_API_KEY` - 新浪财经 API 密钥（用于获取官方公告）
  - `HUIBO_API_KEY` - 慧博投研 API 密钥（用于获取研报）

- **推送服务配置**：
  - `WECHAT_CORP_ID` - 企业微信 Corp ID
  - `WECHAT_APP_SECRET` - 企业微信应用密钥
  - `WECHAT_AGENT_ID` - 企业微信应用 ID

- **监控设置**：
  - `MONITORING_STOCKS` - 监控的股票代码列表

## 使用方法

### 运行应用

```bash
python main.py
```

应用会自动执行以下流程：

1. 采集信息（官方公告、市场数据、研报、行业信息）
2. 清洗和去重信息
3. 分类信息
4. 深度分析各类信息
5. 多维度推理评估机会和风险
6. 根据评分做出决策
7. 生成研报简报
8. 推送到指定渠道

### 查看输出

- **控制台输出**：实时显示执行过程和结果
- **企业微信**：收到推送的研报简报
- **日志文件**：记录详细的执行日志

## 核心优势

1. **效率提升**：每天节省 2-4 小时信息筛选/分析时间，聚焦核心决策
2. **风险规避**：提前预警财务雷、监管雷，避免踩坑
3. **机会捕捉**：快速抓住政策/业绩/资金驱动的机会，不缺席主线
4. **体系化**：把投资逻辑固化为 Agent 工作流，避免情绪化决策
5. **工业级可靠性**：异常处理和容错机制确保系统稳定运行

## 注意事项

- **API 密钥安全**：确保 `.env` 文件不会被提交到版本控制系统
- **合规性**：爬取数据仅限公开信息，避免使用非合规数据源
- **时效性**：A 股交易时间（9:30-11:30/13:00-15:00）需实时采集，非交易时间采集公告/政策
- **本土化**：LLM 提示词已适配 A 股术语（如“北向资金”“龙虎榜”“定增”）

## 部署建议

### 生产环境部署

1. **服务器选择**：选择稳定的云服务器，确保网络连接良好
2. **定时执行**：使用 crontab 或 Windows 任务计划程序定时运行
3. **监控告警**：设置系统监控，确保应用正常运行
4. **数据备份**：定期备份配置文件和重要数据

### 性能优化

- **并行处理**：对于耗时的信息采集和分析任务，考虑使用并行处理
- **缓存机制**：对频繁访问的数据进行缓存，减少 API 调用
- **资源管理**：合理设置 LLM 调用频率，避免超过 API 限制
